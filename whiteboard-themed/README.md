# Whiteboard (React + TypeScript)

A small, polished whiteboard app built with **React**, **TypeScript**, and **Vite**.

## Features
- Pen + eraser
- Color + brush size
- Undo / redo (Ctrl/Cmd+Z, Ctrl/Cmd+Y)
- Clear board
- Export to PNG
- Autosave to localStorage
- Retina / high-DPI crisp canvas

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
